dialog.js使用教程

一、基本使用方法

安装：

$ npm init

$ npm install new_dialog

引入：

<link rel="stylesheet" href="./node_modules/new_dialog/dialog.css">

<script src="./node_modules/new_dialog/index.js"></script>

使用：

new Dialog({ type:"info", //对话框类型，info、success、warning、danger四类，可以扩展后面有介绍 text:"示例一", //对话框内容，默认：hello world! title:"hello world!", //对话框标题 context:"", //对话框上下文，后面有介绍 style:{ //对话框样式 width : "400px", //宽度 height : "100px", //高度 left : "400px", //定位X top : "300px" //定位Y } });

二、扩展

添加自定义类（class）：

这里使用animate.css进行演示

new Dialog({ type:"success", text:"示例二", title:"hello world!", extendClass:"animate animate-shake",//扩展自定义类 style:{ width : "400px", height : "100px", left : "400px", top : "300px" } });

示例：
扩展对话框类型：

扩展对话框类型，只需要为type属性赋值为自定义的值如：mytype,然后在css文件中定义.dialog-mytype{}

js文件

new Dialog({ type:"mytype", //自定义类型 text:"示例三", title:"hello world!", extendClass:"", style:{ width : "400px", height : "100px", left : "400px", top : "300px" } });

css文件

.dialog-mytype{ color: #8a6d3b; background-color: #fcf8e3; border-color: #faebcc; }

示例：
为对话框指定上下文：

为对话框指定上下文，只需要为context属性指定上下文

注意：上下文必须是定位元素，要不没有意义，而且指定上下文后style属性里的top、left都相对于上下文

js文件

new Dialog({ type:"mytype", //自定义类型 text:"示例四", title:"hello world!", extendClass:"", context:document.body, //指定上下文 style:{ width : "400px", height : "100px", left : "400px", top : "300px" } });
